## Getting Started

1. **Install Helm**  
   Get the latest [Helm release](https://github.com/kubernetes/helm#install).

2. **Add YOURLS Helm Chart repository**  
   ```console
   helm repo add yourls https://charts.yourls.org/
   helm repo update
   ```

3. **Install a chart**  
   See [charts respective READMEs](charts).
